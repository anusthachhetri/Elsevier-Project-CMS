Content Profile Patterns for Content and Data

Author

Written by: F. Gordon (f.gordon@mongo.planet)

Overview

Content Profile is a new format for bringing content and data together in an open and flexible manner. The Content Profile format establishes a standard way to express content and data together and is intended to become the common format used for content that we author and for content that we acquire from third-parties. Documents that use the format are called Content Profile Documents.

Components

Content Profile makes use of the industry standard formats HTML and Linked Data. XHTML5 is used as the base fabric for capturing content, while JSON-LD is used to capture data about the content, about various parts of the content, and to capture relationships between the content and other data resources. The JSON-LD packages a set of RDF statements (subject-predicate-object) in a JSON format.
